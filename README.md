# Pingbomb Bot

A modular Discord bot built with Node.js and discord.js.  
This bot includes a controlled pingbomb feature with safety, logging, cooldowns, and owner controls.

---

## Features

- Slash commands
- Pingbomb with multi-target support
- Stop / StopAll controls
- Cooldown system
- Persistent state handling
- Structured logging
- Modular codebase (easy to maintain)
- Ready for 24/7 hosting (Railway / Render)

---

## Tech Stack

- Node.js
- discord.js
- dotenv
- GitHub
- Railway (deployment)
